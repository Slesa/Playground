NetSparkle - AutoUpdate for .NET Developer
------------------------------------------
 
Copyright (c) 2010 Dirk Eisenberg

NetSparkle is an easy-to-use software update framework for .NET developers on Windows, 
MAC or Linux. It was inspired by the Sparkle project for Cocoa developers and the 
WinSparkle project (a Win32 port).
