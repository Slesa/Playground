\section{User Management System}

\subsection{Einleitung}
Das Modul UMS kümmert sich um die Benutzerverwaltung des Systems. Es können Benutzer (User) angelegt und bestimmten Benutzerrollen (UserRole)
zugeordnet werden.



\subsection{Modell}


\subsubsection{UserRole}
Beschreibt eine Benutzerrolle:
\begin{itemize}
	\item Name (nn) Name der UserRole
\end{itemize}


\subsubsection{UserRoleItem}
Beschreibt einen Eintrag in einer Benutzerrolle
\begin{itemize}
	\item Program Betroffenes Programm
	\item Module Modul innerhalb des Programms
	\item Function Funktion innerhalb des Moduls
\end{itemize}


\subsubsection{User}
Ein (möglicher) Benutzer des Systems.
\begin{itemize}
	\item Name (nn) Name der User
\end{itemize}
